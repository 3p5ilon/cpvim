# Vim Configuration

This folder contains my Vim / MacVim configuration.

## Plugins

This setup uses [Vim-Plug](https://github.com/junegunn/vim-plug) as the plugin manager.

The following plugins are used:

- [UltiSnips](https://github.com/SirVer/ultisnips) – snippet engine
- [Jellybeans](https://github.com/nanotech/jellybeans.vim) – Vim color scheme
- [vim-commentary](https://github.com/tpope/vim-commentary) - comment stuff out
